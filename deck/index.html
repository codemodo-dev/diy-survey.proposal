Platform Overview

Apa yang akan kami bangun?

Tiga area utama yang terintegrasi dalam satu sistem — masing-masing melayani role yang berbeda.

🎨

Admin Panel & Survey Designer

Interface untuk Super Admin, Admin, dan Client — manage user, build kuesioner dengan drag-and-drop, set logic, quota, targeting, publish survei, dan monitor fieldwork.

Super Admin · Admin · Client
🌐

Survey Portal

Portal khusus panelist — registrasi, lengkapi profil demografis, browse survei yang tersedia, isi survei, dan terima insentif otomatis setelah complete.

Panelist
📊

Analytics & Disbursement

Dashboard analitik real-time built-in di Survey Designer — chart, filter by segment, export CSV. Plus pencairan insentif panelist via Xendit terintegrasi.

Client · Admin

Pendekatan Kami

Dibangun
phase per phase,
terukur & terstruktur.

Setiap phase menghasilkan deliverable yang bisa diterima, ditest, dan diapprove sebelum lanjut ke phase berikutnya. Tidak ada biaya yang keluar tanpa hasil nyata.

🔧

Multi-tenant Architecture

Setiap client terisolasi datanya.

📝

Fixed Scope per Phase

Scope setiap phase terdefinisi jelas. Perubahan di luar scope dikenakan biaya terpisah.

✅

Acceptance per Phase

14 hari kerja acceptance period setiap phase. Review, test, dan approve sebelum termin berikutnya berjalan.

📦

Deliverables Lengkap

Setiap phase disertai dengan review design dan UAT. Kemudian diakhir phase kami serahkan source code, user guide, FAQ, dan Figma file.

User Roles

Siapa yang akan menggunakan platform ini?

Empat role dengan akses dan kapabilitas berbeda — terisolasi per tenant, dikelola lewat satu sistem.

🔐

Super Admin

  • User management
  • Permission configuration
  • Client data management
  • Platform monitoring
  • Audit log akses penuh
⚙

Admin

  • Akses sesuai permission
  • Monitor panel list
  • Manage client data
  • Set insentif survei
  • View platform activity
🏢

Client

  • Build & publish survei
  • Set targeting & quota
  • Monitor fieldwork
  • Analitik & export data
  • Manage workspace member
👤

Panelist

  • Registrasi & profil demografis
  • Browse & isi survei
  • Eligibility check otomatis
  • Terima insentif otomatis
  • Cairkan ke rekening

08 / 10
Out of Scope

Yang tidak termasuk dalam pengerjaan ini.

Bukan bagian dari kontrak — perlu diketahui sejak awal untuk menghindari ekspektasi yang salah.

📵

Mobile App

Web only — tidak ada native iOS atau Android app.

🚫

Landing Page

Tidak termasuk halaman marketing atau landing page publik.

🔌

Third-party Integrations

Tidak ada integrasi eksternal di luar yang sudah didefinisikan (e.g. Salesforce, Tableau).

🔑

Public API

Tidak tersedia public API untuk di-consume oleh client secara langsung.

💳

Pricing & Billing

Tidak ada kalkulasi harga, invoicing, atau billing system.

🖥️

Server Production Cost

Biaya server production ditanggung client, tidak termasuk dalam biaya pengerjaan.

📦

Data Migration

Pemindahan data existing milik client ke platform baru di luar scope.

🔧

Ongoing Maintenance

Setelah warranty selesai, maintenance sepenuhnya tanggung jawab client.
